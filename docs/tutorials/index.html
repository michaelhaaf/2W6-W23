1.2 Week 04 Exercises

The first two sets of Odin Project exercises. They are great general review of CSS foundations/the box model if you ever feel the need to practise.

The final set of exercises comes from a MDN Web Docs exercise set . It is comprehensive practise for finding and using external web fonts, as well as perfecting typographic CSS skills.
