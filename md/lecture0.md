# Technologies

## HTML

[HyperText Markup Language](https://en.wikipedia.org/wiki/HTML) is the markup language used to define how data is displayed in a webpage. 

- something in a list {.fragment}
- something else in a list {.fragment}

SVG, XML (therefore .docx), and Markdown are also examples of markup languages.

---

Using HTML, we control the **content** we want to display in an interface. All text, images, links, lists, buttons, etc. are specified as **elements** of an interface using HTML. 

We also control the **structure** of the content using HTML. HTML elements can contain other HTML elements, allowing us to **nest** related content.

## CSS

[Cascading Style Sheet](https://en.wikipedia.org/wiki/HTML) is the language what adds style to those plain elements. HTML puts information on a webpage, and CSS positions that information, gives it color, changes the font, and makes it look great!
